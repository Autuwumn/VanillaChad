# Vanilla Chad Mod

Adds 1 Class and 2 Cards
- Vanilla Chad
	- Gives 10x vanilla card chance
- Vanilla Power
	- Grants 5% compounding stats for every vanilla card you have
	- Currently effects damage, attack speed, health, move speed, and jump height
